# Key Stakeholders, Competitors & Evaluation Tools

## Evaluation Platform Companies

| Platform | Focus | Key Feature |
|----------|-------|-------------|
| **Braintrust** | Rapid prototyping + CI/CD eval | Loop AI automates experiment cycles |
| **LangSmith** (LangChain) | Observability + eval for LangChain/LangGraph | Multi-turn agent evaluation, step-level scoring |
| **Arize AI** | Enterprise observability | $70M Series C; Uber, Booking.com; trillions of events/month |
| **Confident AI / DeepEval** | OSS LLM testing | Pytest-like unit testing with G-Eval, task completion |
| **Maxim AI** | Comprehensive agent eval | Enterprise-grade with comparative analyses |
| **Langfuse** | Open-source tracing | OpenTelemetry integration, no built-in eval |
| **Evidently AI** | ML/LLM monitoring | Drift detection; agent eval still limited |

## Open Source Evaluation Tools

| Tool | Capabilities |
|------|-------------|
| **DeepEval** | Pytest-style LLM unit testing; RAG metrics; CI/CD |
| **Langfuse** | Open-source tracing/observability; session grouping |
| **Arize Phoenix** | Open-source LLM tracing |
| **Bloom** (Anthropic) | Behavioral evaluations of frontier AI models |
| **Microsoft Evals for Agent Interop** | Realistic work scenario evaluation (email, calendars) |
| **BrowserGym / WorkArena** | Universal web task simulation |
| **RAGAS** | RAG-specific evaluation |
| **Phil Schmid's Compendium** | Catalog of 50+ benchmarks |

**Community-recommended stack**: DeepEval for CI/CD tests + Langfuse for production tracing + RAGAS for RAG
